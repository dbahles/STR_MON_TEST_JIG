2. Normal Test Procedure

  1. Power the test jig.
  2. Confirm the Ready LED is slowly flashing. This means no DUT is detected.
  3. Place the STR-MON PCB into the jig pocket. Make sure it sits flat and is aligned with the locating features.
  4. Close the clamp so the DUT is held firmly against the test contacts.
  5. Confirm the Ready LED changes to solid on. The jig should give one short beep.
  6. Press the TEST button once.
  7. During testing, the Pass and Fail LEDs alternate. Do not remove or touch the DUT while testing is running.
  8. Wait for the final LED result and buzzer indication.
  9. Remove the DUT and sort it according to the result.

4. Optional Retest Without Removing DUT

In special cases, the tester may prepare the jig for a second test without removing the DUT.

  1. Wait until the DUT has reached Pass or Fail.
  2. Release the TEST button if it is pressed.
  3. Hold the TEST button for approximately 1.5 seconds.
  4. Confirm the jig returns to Ready with the Ready LED solid.
  5. Press TEST once to run the automated test again.
Note: This retest function is intended for special cases only. Normal use is to remove the DUT after the first final result.

5. Test Sequence

The operator does not need to follow these steps manually; the jig performs them automatically after the TEST button is pressed.

8. Serial Output

The serial terminal is optional. It is intended for engineering debug and high-level test checks only. General testers should use the LEDs and buzzer as the official operating indication.

Normal use: Fit DUT, clamp DUT, press TEST, observe Pass or Fail LED.
